Kiosk Support


GitHub Configuration

Directory Setup.

Your GitHub repository needs to have a directory named "kiosk" that contains at least an "index.html" file at the root.

During your development it's strongly encourged to develop in a standard browser before deploying as GitHub has a public limit of 60 API calls per hour and that can quickly be used when pushing files to the device during development.

Important Note: You can easily increase the GitHub API limit by adding a token that can be generated when logged into GitHub at https://github.com/settings/tokens and adding it in the GitHub configuration page in the Kiosk GitHub settings page.
